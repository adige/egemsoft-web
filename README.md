============

Web app for egemsoft.net

Scaffolding via [yeoman-angular](https://github.com/yeoman/generator-angular) generator.

Install
=======
```bash
$ npm install -g bower grunt-cli 
$ npm install  
$ gem install compass  
$ bower install  
```

Test & Run
===========
```bash
$ grunt test
$ grunt serve
```
