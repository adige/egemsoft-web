config
service in module egemsoftWebApp

Description

Egem web app config provider. Serves configuration variables.

Properties

  • imagesUrl

    • Web site images root url. Static images like logo etc.

  • uploadsUrl

    • Uploaded items url. Page content images etc.

  • contentSrc

    • Page and section contents source directory.

  • sections

    • Home page sections. There has to be section contents with the same name under content directory.

  • home.slideInterval

    • Home page main carousel auto slide interval in milliseconds.

  • home.blocksPerSlide

    • Blocks carousel number of items per slide.

  • home.blocksSlideInterval

    • Home page blocks carousel auto slide interval in milliseconds.

  • home.en.slides

    • Main carousel items for English locale.

  • home.en.blocks

    • Blocks carousel items for English locale.

  • home.tr.slides

    • Main carousel items for Turkish locale.

  • home.tr.blocks

    • Blocks carousel items for Turkish locale.
