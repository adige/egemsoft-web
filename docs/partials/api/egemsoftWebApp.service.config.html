config
service in module egemsoftWebApp

Description

Egem web app config service. Provider to serve configuration variables.

Properties

  • home

    Home page configurations.

  • blocksPerSlide

    Carousel auto slide interval in milliseconds.

  • blocksSlideInterval

    Home page blocks carousel auto slide interval in milliseconds.

  • en

    Home page English locale bound configs.

  • blocks

    Blocks carousel items for English locale.

  • slides

    Main carousel items for English locale.

  • slideInterval

    Carousel auto slide interval in milliseconds.

  • tr

    Home page English locale bound configs.

  • blocks

    Blocks carousel items for Turkish locale.

  • slides

    Main carousel items for Turkish locale.
